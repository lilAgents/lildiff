# lilDiff

Text Difference Checker

Compare two versions of text and see exactly what changed: every addition and deletion, by word, line, or character. Built for copy edits and contract revisions.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
